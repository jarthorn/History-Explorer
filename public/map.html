Canadian National Historic Sites

The map below shows all designated federal national historic sites in Canada. Click or touch a pin on the map to learn more details about the site. Larger pins are sites that are either maintained by Parks Canada or open as a museum. Note that some historic sites are private property and may not be accessible to the public.

About this site

This site was built as an example of using open data to power a simple interactive web application. You can see the source code on GitHub. Contributions welcome! If you find a problem with the data, you are also welcome to open an Issue on GitHub.

The data source is an open data set published by the Government of Canada. Visit Canada's open government portal at open.canada.ca to learn more about Canada's Open Data initiatives. Data is licensed under the Open Government Licence - Canada

This application is hosted on the IBM Bluemix cloud platform.
